A tool to help associations to manage registrations for events.

== Description ==
Did you ever tried to plan an event, like a camping trip or a meeting?
With this tool you can allow your participants so signup in an easy way.

It also manages the participation fees and can remember participants to pay, if they did not.

You are also able to create important information like lists of allergies, information about vegan or vegetary stuff
for the kitchen crew and also important first-aid lists.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/solea` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
